Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface that represents a document (like HTML) as a tree of nodes, allowing scripts to dynamically manipulate its structure, style, and content.

How do you select an element from the DOM?

we can select elements using methods like:

document.getElementById('id')
document.getElementsByClassName('className')
document.getElementsByTagName('tagName')
document.querySelector('selector')
document.querySelectorAll('selector')


What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events from its child elements, rather than adding separate event listeners to each child. This is possible because of event bubblingâ€”when an event is triggered on an element, it bubbles up through its ancestors in the DOM tree.

How do you manipulate an element's attributes and styles using the DOM?

Attributes:
Use getAttribute(),
setAttribute(),
and removeAttribute().
Styles:
Use element.style.property = 'value' for inline styles or
element.classList.add('className') to manipulate CSS classes.
