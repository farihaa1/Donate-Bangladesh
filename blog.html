What is the Document Object Model (DOM)?

The DOM is a programming interface that represents a document (like HTML) as a tree of nodes, allowing scripts to dynamically manipulate its structure, style, and content.

How do you select an element from the DOM?

ou can select elements using methods like: document.getElementById('id') document.getElementsByClassName('className') document.getElementsByTagName('tagName') document.querySelector('selector') document.querySelectorAll('selector')

How do you manipulate an element's attributes and styles using the DOM?

Attributes: Use getAttribute(), setAttribute(), and removeAttribute(). Styles: Use element.style.property = 'value' for inline styles or element.classList.add('className') to manipulate CSS classes.
